# Plan Web Corporativa: Kleyson Klaus

## Objetivo
Construir una landing corporativa de una sola página para la marca Kleyson Klaus. La web debe presentar a la empresa con una identidad neutra y profesional, mostrar aplicaciones ya publicadas en Play Store y App Store, y ofrecer formas claras de contacto.

## Principios de marca
- Kleyson Klaus se trata como una marca abstracta, sin relación con videojuegos, franquicias o referencias culturales prestadas.
- La dirección visual debe sentirse tecnológica, sobria y confiable.
- La comunicación debe ser simple, clara y orientada a producto.

## Alcance de esta iteración
- Hero principal con propuesta de valor.
- Sección de empresa o posicionamiento.
- Sección de aplicaciones publicadas.
- Sección de proceso o forma de trabajo.
- Sección de contacto.
- Footer con navegación resumida.

## Fuera de alcance por ahora
- Backend.
- CMS o panel administrativo.
- Formulario conectado a un servicio real.
- Integración con APIs.
- Multiidioma.
- Sitio multipágina.

## Decisiones técnicas
- Mantener HTML, CSS y JS vanilla.
- Crear una hoja nueva de estilos principal para no depender de la portada anterior.
- No reutilizar el fondo ni los assets visuales actuales como base de marca.
- Dejar los datos de apps preparados para reemplazo futuro por información real.

## Estructura objetivo
1. Header fijo con navegación por anclas.
2. Hero con propuesta de valor y métricas.
3. Sección de posicionamiento de la empresa.
4. Sección de aplicaciones con tarjetas.
5. Sección de metodología o flujo de trabajo.
6. Sección de contacto.
7. Footer.

## Criterios de aceptación
- La página debe funcionar correctamente con scroll largo.
- El diseño debe verse bien en móvil y escritorio.
- No debe haber referencias visuales o textuales a StarCraft.
- Las aplicaciones deben quedar preparadas como tarjetas reutilizables.
- El contenido debe sonar a empresa, no a portafolio personal.
