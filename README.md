# web_kleysonklaus

Landing corporativa en HTML y CSS vanilla para la marca Kleyson Klaus.

## Estructura actual
- `index.html`: entrada principal del sitio.
- `styles/`: estilos del frontend publicado.
- `assets/`: recursos visuales del sitio.
- `cgi-bin/`: carpeta reservada para hosting o compatibilidad.
- `_workspace/`: documentacion operativa, arquitectura y reglas para agentes.

## Documentacion operativa
La documentacion de trabajo ya no vive en carpetas del producto. Todo el material para agentes, auditorias y reglas se encuentra en `_workspace/`.

- `_workspace/README.md`
- `_workspace/guides/`
- `_workspace/rules/`

## Archivos protegidos
No deben modificarse ni eliminarse sin aprobacion explicita del usuario:

- `.cpanel.yml`
- `.htaccess`
- `app-ads.txt`
